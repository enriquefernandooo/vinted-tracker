# Vinted Tracker

Preisanalyse & Inventory Tool für Vinted Reselling.

## Features
- Preisfinder: Analysiert echte Vinted-Listings und gibt eine Preisempfehlung
- Inventory Tracker: Artikel mit EK/VK und Marge tracken
- Datenbank: Alle Daten gespeichert in Supabase

## Tech Stack
- Python, Streamlit, Supabase, BeautifulSoup
